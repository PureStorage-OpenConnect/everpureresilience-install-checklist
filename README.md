# Everpure Resilience service - Installation Checklist
Everpure Resilience service Pre-Installation Checklist

## Overview
The checklist is used to gather data that is required as part of a deployment and configuration of sites. 

## Usage
The file is a spreadsheet provided in XML format.  It has been successfully used and tested with Microsoft Excel.
